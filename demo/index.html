Алгоритм нахождения корней:

  • Находим дискриминант: дискриминант равен бе в квадрате минус четыре а це.$D = b^2-4ac$
  • Находим корни: $$ \begin{cases} x_{1,2}=\dfrac{-b\pm\sqrt{D}}{2a} &, \text{если } D > 0 \\ x=\dfrac{-b}{2a} &, \text{если } D = 0 \\ \text{Корней нет} &, \text{если } D < 0 \\ \end {cases} $$ икс первое и второе равно дроби, в числителе минус б плюс минус корень из дискриминанта, в знаменателе два а, при условии, что дискриминант положительный икс равен минус б деленное на два а, при условии, что дискриминант равен нулю корней нет, при условии, что дискриминант отрицательный
Находим дискриминант по формуле: $D = b^2-4ac$.
Если дискриминант положительный, то корни уравнения находятся по формуле: $x_{1,2}=\dfrac{-b\pm\sqrt{D}}{2a}$.
Если дискриминант равен нулю, то корень второй кратности находится по формуле: $x=\dfrac{-b}{2a}$.
Если дискриминант отрицательный, то корней в поле действительных чисел нет.
