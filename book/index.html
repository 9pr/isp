Заголовок презентации. Рекомендации.

Заголовок слайда. по созданию презентаций.

Для перемещением между слайдами следует пользоваться клавишами <Space> <Left> и <Right>. Для включения субтитров, используйте клавишу <D>

Конец слайда, для продолжения нажмите <Space>

Заголовок слайда. Для людей с проблемами зрения.

  • Придерживайтесь стандарту WIA-ARIA(ToDo);
  • Описывайте все медиаобъекты
    • Изображения при помощи свойства alt
      • 
									<img src="link" alt="Описание">
    • Визуальные объекты, такие как например выделения, при помощи свойства title
      • 
									<span class="predicate" title="Сказуемое">Работает</span>
Конец слайда, для продолжения нажмите <Space>

Заголовок слайда. Для людей с проблемами зрения.

  • Продумывайте легенду, чтобы любой незрячий, впервые попав на презентацию, смог с легкостью по ней перемещаться. Описывайте на первом слайде всю навигацию. 
    
							<span class="offscreen legend">
							  Для перемещением между слайдами следует пользоваться
							  клавишами <kbd><Space></kbd>  <kbd><Left></kbd>
							  и <kbd><Right>.</kbd>
							  Для включения субтитров, используйте клавишу <kbd><D></kbd> 
							</span>
Конец слайда, для продолжения нажмите <Space>

Заголовок слайда. Для людей с проблемами зрения.

  • Используйте класс offscreen для любой информации, которую необходимо проговорить скринридеру, но которую не нужно показывать на слайде. Например, описание объекта или знаки препинания. 
    
							<h3><span class="offscreen">Заголовок слайда. </span>
							  Р<span class="offscreen">.</span></h3>
Конец слайда, для продолжения нажмите <Space>

Заголовок слайда. Для людей с проблемами зрения.

  • К видео всегда вставляйте субтитры для тифлокомментариев
  • Для паузы скринридера, если визуально знаки препинания не выделены, необходимо проставить их принудительно. Например, в конце заголовка, при помощи класса offscreen
    • 
							<span class="offscreen">.</span>
Конец слайда, для продолжения нажмите <Space>

Заголовок слайда. Для людей с проблемами зрения.

  • Сам класс offscreen необходимо описать следующим образом:
    • 
							.offscreen {
								clip: rect(1px 1px 1px 1px);
								clip: rect(1px, 1px, 1px, 1px);
								height: 1px; width: 1px;
								overflow: hidden; position: absolute;
								white-space: nowrap;
							}
							.offscreen:focus {
								position:relative;
								clip:auto; overflow:auto;
								width:auto; height:auto;
							}

Заголовок слайда. Для иноязычных людей.

  • При почти нулевой аудитории создание целой новой презентации нецелесообразно и рекомендуется вставить виджет Yandex переводчика или других сервисов.
    • 
								<div style="position: fixed; bottom: 0; left: 140px; 
									z-index: 9999;" id="ytWidget"></div>
								<script src="https://translate.yandex.net
									/website-widget/v1/widget.js?widgetId=ytWidget&
									pageLang=ru&widgetTheme=light&autoMode=false" 
									type="text/javascript">
								</script>

Заголовок слайда. Для людей с проблемами верхних конечностей.

  • Для доступности людей с проблемами конечности необходимо подключать модули голосового управления. (В разработке)
  • Не блокировать и не переопределять стандартную клавитуарную навигацию, принятую во многих программах для просмотра презентаций.

Заголовок слайда. Для людей с проблемами слуха.

  • Необходимо прописывать полную версию слайда в заметки, чтобы слушатель смог прочитать нераспознаную лектором информацию
  • Все аудиоряды должны сопровождаться субтитрами
